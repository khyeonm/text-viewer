# text-viewer

Syntax-highlighted text viewer with line numbers, word wrap toggle, and search.

## Features

- Syntax highlighting via highlight.js (Python, R, JSON, YAML, XML, Shell, Markdown, etc.)
- Line numbers with click-to-highlight
- Word wrap toggle
- In-file text search with match count
- File size and line count summary
- Auto-detects language from file extension
- Monospace font rendering

## Supported Extensions

- `.txt`, `.log`
- `.json`, `.yaml`, `.yml`, `.xml`, `.toml`, `.ini`, `.cfg`
- `.md`
- `.sh`
- `.py`, `.r`, `.R`
- `.nf`, `.smk`

## Installation

Install from the **Plugins** tab in the AutoPipe desktop app.
